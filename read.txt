Execution Steps:-
---------------
1)Copy the package as-is on your desktop
2)Open the IDE(I used Pycharm)
3)Execute pip install requirements.txt in the terminal
4)Execute pipeline.py

Testing:-
-------
In order to check if the tables are populated, Please follow the below steps :-
1)Open cmd
2)change directory(cd) to the package folder till db i.e C:\\Users\\XXXX\\XXXX\\db
3)Type sqlite3
4)Against the sqlite prompt, Type .open abcdb.db
5)Type .tables (the list of tables we created will be displayed) 
6)Do select * from any table to check if the data is populated

Functionality of each script:-
----------------------------
paths.py - Fetches the current directory and creates the target folder db where sqlite3 db file will be created
SQLite_Table_Creation.py - Creates the db file "abcdb.db" in the db folder with the DDLs of 4 dimensions and 1 fact
DataIngest.py - Ingests the data from the csv file to the tables
